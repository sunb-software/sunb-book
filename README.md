# 开发环境

建议开发者使用以下环境，这样避免版本带来的问题

* IDE:eclipse、idea
* DB:Mysql5.7
* JDK:JAVA 8
* WEB:Tomcat8

# 运行环境

* WEB服务器：Weblogic、Tomcat、WebSphere、JBoss、Jetty 等
* 数据库服务器：Mysql8.1
* 操作系统：Windows、Linux、Unix 等
